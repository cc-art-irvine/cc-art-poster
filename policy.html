课程政策

请仔细阅读以下政策条款

📝 报名政策

试课安排

  • • 首次学员需参加$20试课，体验课程内容和教学方式
  • • 试课包含所有材料费用，无额外收费
  • • 试课后可决定是否购买正式课程套餐
  • • 试课费用不可抵扣正式课程费用

年龄要求

  • • 建议年龄：3岁及以上
  • • 3-5岁儿童需家长陪同
  • • 成人课程欢迎零基础学员
  • • 特殊需求学员请提前沟通

课包规则

  • • 课包购买后有效期为12个月
  • • 课包不可转让给他人（同一家庭兄弟姐妹除外）
  • • 课包余额不可提现
  • • 过期课包恕不延期

📅 预约与取消政策

课程预约

  • • 需至少提前24小时预约课程
  • • 可通过微信、电话或网站预约
  • • 预约成功后会收到确认信息
  • • 建议定期预约以确保上课连续性

取消课程

  • • 需至少提前24小时取消课程
  • • 24小时内取消将扣除一次课时
  • • 无故缺席将扣除一次课时
  • • 紧急情况可特殊处理（需提供证明）

改期规则

  • • 可免费改期一次（需提前24小时）
  • • 同一课程多次改期需支付$10手续费
  • • 改期课程需在原课程时间后30天内完成
  • • 特殊情况下可协商处理

🌤️ 天气政策

恶劣天气

  • • 雨天、雪天或极端天气将自动改期
  • • 老师会提前2-4小时通知学员
  • • 可免费改期至合适时间
  • • 如学员坚持上课，将安排室内替代课程

微雨天气

  • • 小雨天气下可选择继续上课（有遮蔽处）
  • • 学员可选择改期，不收取任何费用
  • • 课程会根据天气调整内容
  • • 安全第一，最终决定权在老师

🛡️ 安全政策

学员安全

  • • 户外课程选址确保安全性
  • • 课程区域禁止无关人员进入
  • • 建议购买个人意外保险

健康要求

  • • 学员身体不适时请勿上课
  • • 有传染性疾病时需居家休息
  • • 特殊病史需提前告知老师
  • • 过敏体质请提前说明

材料安全

  • • 使用无毒环保颜料
  • • 画笔等工具定期消毒
  • • 禁止将材料放入口中
  • • 课后需清洁双手

💰 退费政策

试课退费

  • • 试课费用概不退还
  • • 如因老师原因无法上课，可全额退款
  • • 如因天气原因，可免费改期或退款

课包退费

  • • 购买后30天内，未使用课时可申请退款
  • • 已使用课时按单节课价格计算
  • • 退款金额 = 课包总价 - 已使用课时×单节价
  • • 退款将在7-14个工作日内处理

特殊情况

  • • 搬家或长期出差可协商处理
  • • 重大疾病或意外可特殊考虑
  • • 需提供相关证明文件
  • • 最终解释权归CC Art所有

📋 其他条款

照片使用权

  • • 课程期间可能拍摄照片用于教学记录
  • • 照片可能用于宣传推广（可选择退出）
  • • 学员作品照片归学员所有
  • • 如不同意拍摄请提前告知

责任限制

  • • CC Art承担教学期间的基本安全责任
  • • 学员个人物品遗失不承担责任
  • • 因学员个人原因造成的意外不承担责任
  • • 建议购买个人意外保险

政策更新

  • • CC Art保留政策更新的权利
